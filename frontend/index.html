Performance Characteristics

The Vision Transformer structure analyzes localized patterns. Deepfake generation pipelines usually introduce boundary imperfections, blending seam artifacts, or frequency anomalies. The self-attention matrix captures long-range dependencies, tracing inconsistent textures across distant regions (e.g. eyes mismatched, hair blending artifacts) that are imperceptible to human eyes.

  • Hardware Acceleration: CUDA auto-enabled.
  • Inference Delay: ~1.2s on CPU / ~0.15s on GPU.
  • Robustness: High validation score against JPEG compression up to 85% quality.

Vision Transformer (ViT) Architecture Details

This final year project utilizes the ViT-Base transformer architecture, adapted from the landmark paper "An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale".

1
Patch Extraction

An image is split into non-overlapping patches (size 16x16 pixels). For a 224x224 input, this yields exactly 196 patches.

2
Linear Projection

Each patch is flattened and linearly mapped to a D-dimensional embedding vector, matching the transformer's width.

3
CLS Token & Positional Encoding

A learnable [CLS] token is appended to represent the classification output. 1D Positional Encodings are added to maintain spatial orientation.

4
Self-Attention Layers

The sequence of 197 tokens passes through 12 Multi-Head Attention Blocks. This allows patches to relate to all other patches across global distances.
